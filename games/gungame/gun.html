Gun Make Die

Project Type: Game Jam

Game Type: Video Game

Software: Unity, Visual Studio

Coding Languages: C#

Gun Make Die was made in 2 days for the 2022 GMTK Game Jam. The theme was "a roll of the dice" I interpretted the theme very literally and created a round-based first person shooter where the difficulty of each round is determined by the role of a die. The Die in the middle of the level is rolled using physics. When the button is pressed a force is applied and it spawns enemies based on what number comes up. This game includes power ups such as faster firing speed and dual wielding.
