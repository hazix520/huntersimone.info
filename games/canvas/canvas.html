Project Type: Interactive Web Pages

Software: Adobe Dreamweaver

Coding Languages: JavaScript, HTML, CSS

This section contains a variety of interactive projects made in HTML Canvas. Most of tese projects use a coordinate system to recieve input and use that to create images you can interact with. Later, I used sin, cos and tan functions to make this motion automatic. All of the projects in this section were made for my Creative Coding course at the University of Tampa in Spring 2022. I created the original version of "Wanted" for teh capstone of this course.
