Unmanned

Project Type: School Assignment

Game Type: Video Game

Software: Unity, Visual Studio, Blender

Coding Languages: C#

Unmanned is a game where you play as a space rover whose goal is to repair and recover data from abandoned space rovers before coming one itself

  • Hybrid Movement System (Physics and Rotating Planet)
  • Minigame logic
  • 3D Model



Reflection

What is this project?

Unmanned was a solo class project about a space rover that was sent to repair and extract data from other broken down space rovers before it became one of them. (the orbs are placeholders for broken down rovers and the minigames are you fixing them).

My role

This project was made entirely solo. I did all the 3D art, programming and game design. (the planet texture and skybox are free assets. I did not make them)

What I accomplished

I was able to complete the primary mechanics for a simple project with solid, intentional code with no known exploits.

What I Learned

For this project I set the unhelpful goal of making a game with no bugs. The good news is I made something that came very close to that, but unfortunately because I was focusing on making the systems and mechanics perfect I forgot to make the game fun. I saw this project as more of a programming challenge than a design one and that shows in the final product. In the future I would care less about making the systems perfect until I went through rigorous playesting to confirm that the game I am making is indeed fun.
