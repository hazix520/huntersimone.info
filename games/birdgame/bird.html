The Talons Of Icarus

Project Type: Personal Project

Game Type: Video Game

Software: Unity, Visual Studio, Aseprite

Coding Languages: C#

The Talons of Icarus is a game where you play as a baby phoenix that is trying to reach they sky.

  • Physics-based Movement System
  • Combat System
  • Timer and Highscore Tracking
  • Camera Movement and Effects
  • Pixel Art and Animations

Reflection

What is this project?

The Talons of Icarus is a 2D action platformer game inspired by games like Hollow Knight, Titanfall 2, and Cuphead. It was originally planned to be an adventure game, but after some deliberation I decided to turn it into a speedrunning game. You can play this game right now by clicking here

My role

This project was made entirely solo, including all visuals, pixel art, programming and game design.

What I accomplished

After working on this game in my free time for about four months I made a game demo that was played by over 200 people. I was able to collect a lot of valuable feedback and gain the experience of releasing a game.

What I Learned

I learned a lot from my experience developing TTOI. This was my first time showing something I made to a significant audience and I learned the value of playtesting, collecting feedback and how nerve racking it is to have your project criticized. I also learned many of the complications associated with making a pixel perfect game and some tricks to make 2D movement feel really responsive.
