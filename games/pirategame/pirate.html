A Pirate's Life

Project Type: School Project

Game Type: Board Game

This is a board game I developed in a group for my game design class. I acted as the group leader and was the primary contributor for all of the gameplay elements. We worked both in and out of class to refine these elements through playtesting using the iterative design process.

The assignment was to modify the original checkered game of life, make it our own and deliver a physical copy
